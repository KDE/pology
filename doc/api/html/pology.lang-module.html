Package pology :: Package lang

Package lang

Language-specific tools, arranged in submodules by language code.

Check the respective language submodules for the credits and license notes.

Submodules
  • pology.lang.es: Hooks specific for Spanish.
    • pology.lang.es.remove_subs: Remove special substrings from parts of the message.
    • pology.lang.es.remove_tags_without_translation: Remove contexts that do not have need of translation.
  • pology.lang.fr: Language-specific tools for French language.
    • pology.lang.fr.patternAccents: Accent equivalence in regular expression patterns.
    • pology.lang.fr.patternEntities: Convert special entities in rule patterns.
  • pology.lang.ja: Language-specific tools for Japanese language.
    • pology.lang.ja.katakana: Retain only Katakana words in the text, separated by spaces.
  • pology.lang.nn: Language-specific tools for Norwegian Nynorsk.
    • pology.lang.nn.exclusion: Catch inofficial ortography forms in Norwegian Nynorsk translation.
  • pology.lang.sr: Language-specific tools for Serbian language.
    • pology.lang.sr.accents: Process letter accents in Serbian Cyrillic text.
    • pology.lang.sr.charsets: Conversions between character sets in Serbian texts.
    • pology.lang.sr.checks: Various checks for translations into Serbian.
    • pology.lang.sr.nobr: Equip text with no-break characters where possibly helpful.
    • pology.lang.sr.reduce: Reductions of Serbian text convenient in various special uses.
    • pology.lang.sr.trapnakron: Constructors of syntagma derivators for trapnakron.
    • pology.lang.sr.trapres: Resolve trapnakron references in translations.
    • pology.lang.sr.uiref: Additions to resolving UI references.
    • pology.lang.sr.wconv: Conversions between scripts and dialects in Serbian.
