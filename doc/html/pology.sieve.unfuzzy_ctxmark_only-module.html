Module unfuzzy_ctxmark_only

Unfuzzy messages fuzzied only due to a change in context marker.

This sieve is similar to unfuzzy-context-only, but it unfuzzies the message when the only change is in a specific part of msgctxt, in the context marker.

Context marker is a part of KUIT markup (KDE user interface text), which states more formally the general usage intent for the PO entry. For example, there may be two messages which are same in English, but one applied as a menu item, and another as a dialog title; using KUIT, they would be marked as:

   msgctxt "@action:inmenu File"
   msgid "Export as HTML"
   msgstr ""

   msgctxt "@title:window"
   msgid "Export as HTML"
   msgstr ""

The context marker is the leading part of the context, starting with @... and ending with the first whitespace. This sieve will unfuzzy the message if only the context marker has changed (or was added or removed), but not if the change was in the rest of the context, after the first whitespace.

Sieve parameters:

  • noreview: do not add comment about unreviewed context (not advised)
