msgfilter(filtr, options='')

 

Pass PO files through msgfilter(1) [hook factory].

Hooks modify PO files in place; the executed command is:

   msgfilter <options> -i <filepath> -o <filepath> <filtr>

where options parameter may be used to pass any extra options to msgfilter.

Parameters:
  • filtr (string) - filter to use (e.g. cat for no-op)
  • options - additional options to pass to msgfilter @type options string
Returns: (filepath) -> numerr
type F6A hook

Note: In case msgfilter does not finish without errors, hooks always report number of errors as 1.

msgfmt(options='')

 

Pass PO files through msgfmt(1) [hook factory].

The file is not modified; the executed command is:

   msgfilter <options> -o/dev/null <filepath>

where options parameter may be used to pass any extra options to msgfmt.

Parameters:
  • options (string) - additional options to pass to msgfmt
Returns: (filepath) -> numerr
type S6A hook

Note: In case msgfmt does not finish without errors, hooks always report number of errors as 1.
