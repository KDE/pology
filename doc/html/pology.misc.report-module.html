Module report

Report info, warning and error messages.

Functions for Pology tools to report messages to the user at runtime, in different contexts and scenario. May colorize some output.

encwrite(file, text)

 

Write unicode text to file using best encoding guess.

If the file has been opened with explicit encoding, that encoding is used. Otherwise a guess is made based on the environment locale.

Parameters:
  • file (file) - file to write to
  • text (string or unicode) - text to write

report_msg_content(msg, cat, delim=None, force=False, subsrc=None, file=sys.stderr, highlight=None)

 

Report the content of a PO message.

Provides the message reference along with the message contents, consisting of the catalog name and the message position within it.

Parameters:
  • msg (instance of Message_base) - the message to report the content for
  • cat (Catalog) - the catalog where the message lives
  • delim (None or string) - text to print on line following the message
  • force (bool) - whether to force reformatting of cached message content
  • subsrc (None or string) - more detailed source of the message
  • file (file) - output stream. Default is sys.stdout
  • highlight (re.compile regular expression) - regular expression pattern used to colorize text. Default is None
