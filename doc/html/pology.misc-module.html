Package misc

Various functionality for processing contents of PO files.

Authors:
Chusslove Illich (Часлав Илић) <caslav.ilic@gmx.net>, Sébastien Renard <sebastien.renard@digitalfox.org>

License: GPLv3

Submodules
  • pology.misc.colors: Standard codes for terminal colors.
  • pology.misc.comments: Parsing and composing message comments.
  • pology.misc.config: Access user configuration for Pology.
  • pology.misc.diff: Produce special diffs between strings and other interesting objects.
  • pology.misc.entities: Handle entity definitions.
  • pology.misc.escape: Escaping texts in various contexts.
  • pology.misc.fsops: Operations with environment, file system and external commands.
  • pology.misc.langdep: Fetch language-dependent modules, functions, data, etc.
  • pology.misc.markup: Process text markup.
  • pology.misc.merge: Merge PO files.
  • pology.misc.monitored: Framework for monitored classes.
  • pology.misc.msgreport: Report info, warning and error messages.
  • pology.misc.multi: Collections of multiple sequences.
  • pology.misc.normalize: Various normalizations for strings.
  • pology.misc.report: Report info, warning and error messages.
  • pology.misc.resolve: Replacement of segments in text which define some underlying values.
  • pology.misc.rules: Match messages by rules of arbitrary specificity.
  • pology.misc.split: Splitting message fields into syntactical elements.
  • pology.misc.stdcmdopt: Standard command line options.
  • pology.misc.stdsvpar: Standard sieve parameters.
  • pology.misc.subcmd: Handle subcommands and their parameters.
  • pology.misc.synder: Derive forms and properties of syntagmas by macro expansion.
  • pology.misc.tabulate: Pretty-printing of tabular data.
  • pology.misc.timeout: A timeout decorator.
  • pology.misc.vcs: Version control operations.
  • pology.misc.wrap: Text wrapping, with special handling for typical texts in PO files.
