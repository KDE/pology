Package pology :: Package l10n

Package l10n

Language-specific Pology tools, arranged in submodules by language code.

Check the respective language submodules for the credits and license notes.

Submodules
  • pology.l10n.fr: Language-specific tools for French language.
    • pology.l10n.fr.hook: Hooks specific for French.
      • pology.l10n.fr.hook.patternAccents: Accent equivalence in regular expression patterns.
      • pology.l10n.fr.hook.patternEntities: Convert special entities in rule patterns.
    • pology.l10n.fr.sieve: Sieves specific to French language.
      • pology.l10n.fr.sieve.setUbsp
  • pology.l10n.ja: Language-specific tools for Japanese language.
    • pology.l10n.ja.hook: Hooks specific for Japanese.
      • pology.l10n.ja.hook.katakana: Retain only Katakana words in the text, separated by spaces.
  • pology.l10n.nn: Language-specific tools for Norwegian Nynorsk.
    • pology.l10n.nn.hook: Hooks specific for Norwegian Nynorsk.
      • pology.l10n.nn.hook.exclusion: Catch inofficial ortography forms in Norwegian Nynorsk translation.
  • pology.l10n.sr: Language-specific tools for Serbian language.
    • pology.l10n.sr.hook: Hooks specific for Serbian.
      • pology.l10n.sr.hook.accents: Process letter accents in Serbian Cyrillic text.
      • pology.l10n.sr.hook.checks: Various checks for translations into Serbian.
      • pology.l10n.sr.hook.limit_chset: Limit character set in Serbian texts, by conversion and transliteration.
      • pology.l10n.sr.hook.mod_uiref: Post-filter resolved UI references.
      • pology.l10n.sr.hook.nobr: Equip text with no-break characters where possibly helpful.
      • pology.l10n.sr.hook.reduce: Reductions of Serbian text convenient in various special uses.
      • pology.l10n.sr.hook.trapres: Resolve trapnakron references in translations.
      • pology.l10n.sr.hook.wconv: Conversions between scripts and dialects in Serbian.
    • pology.l10n.sr.scripts: Scripts specific for Serbian.
      • pology.l10n.sr.scripts.fallback_import_paths: Insert fallback import paths in scripts for Serbian language.
      • pology.l10n.sr.scripts.pohybdl
      • pology.l10n.sr.scripts.traplint
    • pology.l10n.sr.trapnakron: Constructors of syntagma derivators for trapnakron.
