unescape_strict(s)

 

Strictly unescape text for double-quoted strings.

Exactly reverses result of escape_strict.

Parameters:
  • s (string) - text to unescape (no wrapping double-quotes)
Returns: string

unescape(s)

 

Non-strictly unescape text for double-quoted strings.

This will do the wrong thing upon e.g. \n, but much faster, and usually sufficient.

Exactly reverses the result of escape.

Parameters:
  • s (string) - text to unescape (no wrapping double-quotes)
