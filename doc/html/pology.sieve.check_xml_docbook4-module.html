Module check_xml_docbook4

Lightweight validity checking of POs containing Docbook XML.

Docbook is a complex XML application, and nothing short of full validation of XML files generated from translated POs can show it correct for sure. This sieve just checks for well-formedness and mere existance of a tag in Docbook subset, and on a single PO entry level.
