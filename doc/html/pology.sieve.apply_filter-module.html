Module apply_filter

Apply filters to translation.

Pass msgstr fields through a combination of hooks, of types:

  • F1A ((text)->text) or F3A/C ((text/msgstr, msg, cat)->msgstr), to modify the translation
  • V1A ((text)->spans) or V3A/C ((text/msgstr, msg, cat)->spans), to validate the translation
  • S1A ((text)->spans) or S3A/C ((text/msgstr, msg, cat)->spans), for side-effects on translation (e.g. simpler checks which write notes to standard output, rather than reporting erroneous spans as V* hooks)

Sieve parameters:

  • filter:<hookspec>: hook specification (see misc.langdep.get_hook_lreq for the format of hook specifications)

Parameter filter can be repeated to chain several hooks, which are then applied in the order of appearance in the command line.
