sr_c2l(text)

 

Transliterate Serbian Cyrillic text to Serbian Latin.

Properly converts uppercase digraphs, e.g. Љубљана→Ljubljana, but ЉУБЉАНА→LJUBLJANA.

Parameters:
  • text (string) - Cyrillic text to transform
Returns: string
the text in Latin
