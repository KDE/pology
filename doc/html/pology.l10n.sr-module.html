Package sr

Language-specific tools for Serbian language.

Author: Chusslove Illich (Часлав Илић) <caslav.ilic@gmx.net>

License: GPLv3

Submodules
  • pology.l10n.sr.hook: Hooks specific for Serbian.
    • pology.l10n.sr.hook.accents: Process letter accents in Serbian Cyrillic text.
    • pology.l10n.sr.hook.checks: Various checks for translations into Serbian.
    • pology.l10n.sr.hook.cyr2lat: Transliterate Cyrillic text into Latin.
    • pology.l10n.sr.hook.ijeref: Process iyekavian text with marked yat-reflexes.
    • pology.l10n.sr.hook.mod_uiref: Post-filter resolved UI references.
